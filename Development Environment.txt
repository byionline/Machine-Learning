# Using virtualenv
## 1. Install the following for pip and virtual environment

`sudo apt-get install python3-pip python3-dev python-virtualenv`

## 2. Create virtual environment named 'AI' any name of your choice

`virtualenv -p python3 AI`

## 3. To activate virtual environment

`source AI/bin/activate`

## 4. Your virtual environemt

**(AI) BOT:~$**

## 5. To deactivate virtual environment

`deactivate`

# Install Machine Learning Library in the virtual environment

## 1. Activate the virtual environment

`source AI/bin/activate`

## 2. Check install packages

`pip list--`

## 3. Install the following packages

`pip install numpy scipy pandas matplotlib`

## 4. Install Scikit-learn

`pip install scikit-learn`

## 5. Install Tensorflow for cpu

`pip install tensorflow keras`

There are many more machine learning packages go ahead and try.
